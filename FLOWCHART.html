─── AI Processing Pipeline (what happens on every upload) ───

File uploaded by user

Saved to storage + DB record created → response returned immediately

↓ Background (non-blocking)

1. classifyAndLink()

AI prompt: "Given title + category, which standards does this satisfy?"
→ Creates EvidenceStandardLink rows

2. evaluateChecklist()

AI prompt: "Which of these 28 auditor items does this doc satisfy?"
→ Upserts ChecklistStatus rows

3. evaluateAllRules()

Rule engine (no AI): checks 52 rules against current DB state
→ Updates isPassing + calculates weighted score

Next page load reflects changes

Checklist ticks updated • Standards status updated • Score recalculated
